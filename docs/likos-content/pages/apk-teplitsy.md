## FAQ

- **Щиты монтируются только на каркас теплицы?** — Типовой сценарий для досвета — монтаж на конструкциях тепличного комплекса; возможны иные решения по проекту здания.
- **Требуется ли отдельный щит учёта?** — Учёт электроэнергии выносится в смежные узлы по проекту энергоснабжения объекта.
