## Краткое описание (shortDescription)

Перфорированный каркас и сетка галунного плетения из коррозионностойкой стали для водозабора и мониторинга.

## Полное описание (htmlDescription)

```html
<p>Сетчатый обсадной фильтр состоит из перфорированной обсадной трубы и фильтрующей сетки <strong>галунного плетения</strong> (перекрёстное переплетение проволок для повышения прочности и стабильности ячейки) из коррозионностойкой стали. Перфорация каркаса — щелевая или с круглыми отверстиями; схема и <strong>скважность</strong> согласуются с гидрогеологией.</p>
<p>Конструкция удерживает песок и крупные частицы, снижая износ насосного оборудования. Марка стали сетки и параметры каркаса подбираются по агрессивности воды.</p>
```

## Подбор фильтра (steps)

1. **Диаметр обсадной трубы** — колонна скважины, тип соединений (резьба, муфты).
2. **Дебит** — сопоставление с допустимой скоростью фильтрации и насосом.
3. **Состав воды** — марка нержавеющей стали сетки и каркаса по коррозии.
4. **Условия скважины** — гранулометрия, статический уровень, цементация над фильтром.
5. **Готовое решение** — типоразмер из таблицы или индивидуальный чертёж; монтажные ключи и хомуты — [/categories/likos_of_obustroystvo](/categories/likos_of_obustroystvo).
